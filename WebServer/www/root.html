You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /area -- returns the area of specified shape, ex:/area?width=2&height=3&shape=square
  • /location -- returns the continent of given longitude and latitude, ex:/location?lat=24&long=67
